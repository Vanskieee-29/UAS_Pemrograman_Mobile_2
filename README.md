# MyKeuangan 2.0

MyKeuangan 2.0 merupakan aplikasi pencatatan keuangan pribadi berbasis Android yang dikembangkan untuk membantu pengguna mengelola kondisi keuangan sehari-hari secara lebih mudah dan terstruktur.

Aplikasi ini memungkinkan pengguna untuk mencatat pemasukan dan pengeluaran, memantau saldo, membuat target keuangan, melihat laporan keuangan dalam bentuk visual, serta memperoleh insight berdasarkan aktivitas keuangan yang telah dicatat.

## Fitur Utama

### 1. Beranda (Home)

* Menampilkan total saldo pengguna.
* Menampilkan total pemasukan dan pengeluaran.
* Menampilkan ringkasan kondisi keuangan secara cepat.
* Tampilan modern dengan desain yang lebih responsif.

### 2. Manajemen Transaksi

* Menambahkan data pemasukan.
* Menambahkan data pengeluaran.
* Menyimpan riwayat transaksi secara lokal.
* Data transaksi digunakan sebagai dasar analisis pada fitur lainnya.

### 3. Target Keuangan

* Membuat target keuangan sesuai kebutuhan pengguna.
* Menambahkan progres tabungan terhadap target.
* Menampilkan persentase pencapaian target.
* Notifikasi ketika target berhasil tercapai.

### 4. Laporan Keuangan

* Menampilkan total pemasukan.
* Menampilkan total pengeluaran.
* Menampilkan saldo akhir.
* Menampilkan visualisasi data menggunakan Pie Chart.
* Menampilkan riwayat transaksi berdasarkan tanggal.

### 5. AI Insight

* Menghasilkan Financial Score berdasarkan aktivitas keuangan pengguna.
* Menampilkan status kondisi keuangan.
* Memberikan rekomendasi pengelolaan keuangan.
* Menampilkan insight tag secara otomatis berdasarkan data transaksi.
* Membantu pengguna memahami kondisi keuangan secara lebih cepat.

### 6. Multi Language

* Mendukung Bahasa Indonesia dan Bahasa Inggris.
* Bahasa aplikasi dapat menyesuaikan secara otomatis berdasarkan lokasi pengguna.
* Menggunakan GPS dan Geocoder untuk mendeteksi negara pengguna.

### 7. Bottom Navigation Bar

* Navigasi yang lebih mudah antar halaman aplikasi.
* Terdiri dari menu Home, Transaksi, Target, Laporan, dan AI Insight.

## Teknologi yang Digunakan

* Kotlin
* Android Studio
* SharedPreferences
* Gson
* MPAndroidChart
* Material Design Components
* Fused Location Provider
* Geocoder API

## AI Insight Logic

Fitur AI Insight menggunakan pendekatan rule-based analysis dengan memanfaatkan data transaksi pengguna.

Beberapa parameter yang dianalisis:

* Jumlah transaksi.
* Total pemasukan.
* Total pengeluaran.
* Kondisi saldo pengguna.

Insight yang dapat muncul:

* Pengguna Aktif
* Pengeluaran Tinggi
* Hemat

## Tujuan Pengembangan

MyKeuangan 2.0 dikembangkan untuk membantu pengguna dalam:

* Mencatat aktivitas keuangan sehari-hari.
* Memantau pemasukan dan pengeluaran.
* Mengelola target keuangan.
* Memahami kondisi keuangan melalui laporan dan insight.
* Meningkatkan kesadaran dalam pengelolaan keuangan pribadi.

## Pengembang

Dikembangkan sebagai proyek pengembangan aplikasi Android dengan fokus pada manajemen keuangan pribadi, visualisasi data keuangan, dan pengalaman pengguna yang lebih baik melalui fitur analisis serta dukungan multi-bahasa.
